# 武侠世界(Wuxiaverse) 引擎

Wuxiaverse Engine — Open-source immersive Wuxia world building engine

## 项目简介

Wuxiaverse Engine 是基于 OpenMW 修改和扩展的开源引擎，旨在创建沉浸式武侠世界体验。  
项目遵循 **GPLv3** 开源协议，欢迎社区参与和贡献。

## 占位项目说明

当前仓库为占位版本，后续将逐步公开引擎代码、示例场景和开发文档。
